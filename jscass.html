    let name = prompt ("Please enter your name:"); console. log ("Hello, " + name + "!");

  4. The document write() method deletes all existing HTML when used on a loaded document in javascript
  5. Console. log is a built-in function that allows you to output messages or values to the console.

Define 10 other reserved javascript keyword and what they do.

  1. Null : expresses a lack of identification, indicating that a variable points to no object.
  2. Interface : is a future reserved keyword, it's set aside for use as a keyword in future language extention.
  3. Static : It represent an integer value in JavaScript.
  4. Const : is a keyword used to declare a variable that cannot be reassigned a new value.
  5. Debugger : keyword stops the execution of JavaScript, and calls (if available) the debugging function.
  6. Else : if the same condition is false it specifies the execution for a block of code.
  7. Enum : to represent a fixed set of named values,it allow developers to define a list of named constants,
    which can be used throughout the codebase to increase code readability, maintainability and prevent errors.
  8. Boolean : value is one that can either be TRUE or FALSE. If you need to know “yes” or “no” about something,
    then you would want to use the boolean function.
  9. Instanceof : operator tests to see if the prototype property of a constructor appears anywhere in the prototype chain of an object.
  10. Synchronized : every line of code is executed one after the other, and each task must wait for the previous one to be completed before moving to the next.
